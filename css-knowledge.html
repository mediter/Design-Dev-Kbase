Web Design Knowledgebase - CSS

Universal Selector

* the asterisk symbol, Applies to all elements in the document

Type Selector

HTML tag name. Matches element names

Class Selector

Matches an element whose class attribute has a value that matches the one specified after the period (or full stop) symbol

ID Selector

Matches an element whose id attribute has a value that matches the one specified after the pound or hash symbol

Child Selector

Use the larger than symbol to connect two elements. Matches an element that is a direct child of another

Descendant Selector

Use one space between two elements.Matches an element that is a descendent of another specified element (not just a direct child of that element)

Adjacent Sibling Selector

Use a plus sign between two elements. Matches an element that is the next sibling of another

General Sibling Selector

Use a tilde sign between two elements.Matches an element that is a sibling of another, although it does not have to be the directly preceding element

color

specify the color of text inside an element

color:
[ RGB | RGBA | HEX | Color-Names | HSL | HSLA ]

background-color

specify the background color of an element

background-color:
[ RGB | RGBA | HEX | Color-Names | HSL | HSLA ]

opacity

specify the opacity of an element and any of its child elements.

opacity: ( a number 0.0 ~ 1.0 )

vertical-align

specify the alignment of text on the vertical axis

vertical-align:
[ baseline | sub | super | top | text-top | middle | bottom | text-bottom ]
